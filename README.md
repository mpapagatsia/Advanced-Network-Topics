# Advanced-Network-Topics
This repository contains our implementation for the annual Spectrum Challenge of University of Thessaly.
The challenge was about a USRP-based radio design that could best use a communication channel in the presence of
other dynamic users and interfering signals. 
The challenge took place on the NITOS infrastructure.

We used the GNU Radio as baseline.

The team's name was peer-to-peers.

Our team received the first place on the competition.

Each .txt file contains the changes on the default parameters of the transceiver.

The .py files contain the final implementation.

The implementation consists of: 
 - a bi-directional communication οn the reciever and the transmiter node.
 - transmitter to receiver for the basic transmition of the packets.
 - receiver to transmiter for the feedback channel.
 - compression on the packets.
